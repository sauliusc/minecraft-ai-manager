Under the hood

ALL CUSTOM.

Every mechanic is driven by plugins written specifically for this server. No off-the-shelf bloat — tight, purpose-built code connected to the CraftControl web panel.

Custom Plugins — 13

🔗
BridgePlugin

HTTP gateway between the Minecraft server and the CraftControl web API — player tracking, economy sync, and analytics.

⚔️
ChallengePlugin

Tracks in-game challenge progress and reports completions to the web API for XP and reward dispatch.

🏰
ClanPlugin

Full clan lifecycle — create, join, accept, kick, promote, demote, disband, and list with war support.

🌟
CosmeticsPlugin

Unlockable titles, chat colors, particles, pets, and trails that flex your rank and history.

💰
EconomyPlugin

Dual-currency economy with /pay, balance checks, and a player-driven market. No real-money shortcuts.

🐉
EventPlugin

Boss Raids, Treasure Hunts, and Build Battles — server-wide events with custom mechanics and tiered loot.

👋
GreeterPlugin

Personalised welcome messages and starter rewards for first-time and returning players.

🛡️
ModerationPlugin

Staff tools for chat scanning, player reports, warnings, and bans — surfaced in the web panel.

🧙
NpcPlugin

Guide and quest-giver NPCs with multi-tier dialogue trees — no mods required on the client side.

📜
QuestPlugin

Quest assignment, progress tracking, and completion — daily and weekly objectives with XP and coin payouts.

🎁
RewardPlugin

Delivers mystery-box loot and milestone rewards directly to players — four rarity tiers up to Legendary.

🔥
StreakPlugin

Daily login streak tracking with escalating bonus rewards for consistent players.

🗳️
VotePlugin

Shows voting-site links and automatically grants coin rewards when a player's vote is confirmed.
